# Simple Calculator

## Description

This is a beginner-friendly Python calculator project that performs basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Displays an error message for invalid choices

## Technologies Used

* Python 3

## How to Run

1. Open the terminal or command prompt.
2. Navigate to the project folder.
3. Run the following command:

```bash
python calculator.py
```

## Sample Output

